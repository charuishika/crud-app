# Student Database CRUD App

A simple Node.js + Express + MySQL app for managing student records.

## Features
- Add student
- View all students
- Update student
- Delete student

## Tech Stack
- Node.js
- Express.js
- MySQL (XAMPP)
- Handlebars (hbs)
