# Student Database CRUD App

A simple Node.js + Express + MySQL app for managing student records.

## Features
- Add student
- View all students
- Update student
- Delete student

## Tech Stack
- Node.js
- Express.js
- MySQL (XAMPP)
- Handlebars (hbs)

## Setup
1. Clone the repo  
2. Run `npm install`
3. Copy `.env.example` â†’ `.env` and fill DB details
4. Start MySQL in XAMPP and create DB `charu_app_node`
5. Run the app: `node app.js`

Visit: `http://localhost:5000`
